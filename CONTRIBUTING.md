## Where Contributions Go
Submit contributions to https://github.com/RegardsOss/RegardsOss.github.io as a pull request to the `master` branch.

**All development and pull requests should be done against the `master` branch.**

### Installation

```console
yarn install
```

### Local Development

```console
yarn start
```

This command starts a local development server and open up a browser window. Most changes are reflected live without having to restart the server.

### Build

```console
yarn build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service. It also validates all links in the website are valid.

### Deployment

Once your pull request has been merge into the **master** branch, website is rebuild in few minuts.
