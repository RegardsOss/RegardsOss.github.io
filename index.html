Welcome to the Regards OSS website !

The Regards system is primary designed to store data and their metadata from spacial agency on safe storage and to let scientist order these data.

Harvest any data from anywhere.

Regards can import data from databases (Postgres, Oracle) and files (SIP), but you can develop your own Datasource plugin !

Deep research.

Thanks to ElasticSearch and our modular project interface, we let users dig inside data to order only data they trully need.

Support planets and celestial vault.

Regards display your polygons and others GeoJSON feature directly on a map.
