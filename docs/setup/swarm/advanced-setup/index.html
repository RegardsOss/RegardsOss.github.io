Advanced setup

Deployment#

Introduction#

This section introduce how to deploy a REGARDS stack thanks to ansible on a docker swarm environment.
All REGARDS docker images are available on our github repository.

Pre-requisites#

  1. Install Ansible version 2.6.2 at least docs.ansible.com
  2. Download ansible [regards-playbook]
  3. Extract downloaded ansible regards-playbook zip archive

Deploy custom inventory#

To deploy a custom inventory with some special configuration, you can copy one of the demo or multihost invotory and modify configuration properties.

A framework and a set of plugins to build scientific archives.

REGARDS is an Open Source data valorization product, highly flexible to various themes, modular, efficient, long-lasting, evolutionary and scalable, aims at providing services to Mission Centers and laboratories, and is able to deal with huge amount of data.
Our goal is to allow a strong commitment from user community in the development of modules and plugins.
