Create an order

Allows to validate current basket and create corresponding order.

With label​

Request

URL

/user/orders

URL template

/user/orders

Method

POST

Headers

Authorization:Bearer{token}
Content-Type:application/json;charset=UTF-8
Accept:application/json

labelStringcommand label, 1 to 50 characters, must be unique for current user (generated when not provided)
