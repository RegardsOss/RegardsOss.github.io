REGARDS offre la possibilité de créer plusieurs projets avec leurs propres thématiques.
Depuis l'interface d'administration de REGARDS, vous gérez vos projets et les utilisateurs du système.

:::caution
Cette rubrique est uniquement réservée au `super administrator` défini lors de l'installation du système, il est seul
ayant-droit pour accéder à l'interface d'administration de tous les projets REGARDS.
:::

---

L'interface d'administration de REGARDS est accessible depuis l'adresse `http://<regards host>/admin`.
L'interface d'administration d'un projet est accessible depuis l'adresse `http://<regards host>/admin/<nom projet>`.
L'interface utilisateur d'un projet est accessible depuis l'adresse `http://<regards host>/user/<nom projet>`.
L'interface du portail d'accès est accessible depuis l'adresse `http://<regards host>/`.

Depuis l'interface du portail, vous pourrez :

- [Créer et gérer des projets](project-configuration.md)
- [Identifier les utilisateurs et contrôler leurs droits d'accès](user-accounts.md)
- [Personnaliser l'interface Portal](portal-configuration.md)
